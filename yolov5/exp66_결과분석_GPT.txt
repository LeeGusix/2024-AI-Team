이 이미지에서 YOLOv5의 학습 결과가 표시되고 있습니다. 여기에서 중요한 수치와 그 의미를 설명하겠습니다:

1. 주요 지표 설명
아래 표에서 등장하는 수치는 Object Detection(객체 검출) 모델의 성능을 평가하는 데 사용됩니다.

수치	의미
P (Precision)	검출된 객체 중에서 실제로 정확한 검출의 비율 (정밀도)
R (Recall)	실제 객체 중에서 검출된 객체의 비율 (재현율)
mAP50	IoU=0.5에서의 mean Average Precision. 모델의 정확도를 평가하는 기준입니다.
mAP50-95	IoU=0.5부터 0.95까지 (0.05 간격)의 평균 mAP. 더 엄격한 성능 평가 기준입니다.
Instances	전체 데이터셋에서 객체의 개수입니다.
Images	데이터셋의 이미지 개수입니다.
2. 결과 요약
Images: 총 78,688장의 이미지가 사용되었음을 알 수 있습니다.
Instances: 총 157,377개의 객체가 검출 대상입니다.
각 지표는 다음과 같은 결과를 보여줍니다:

Precision (P): 0.615
→ 검출된 객체 중 약 **61.5%**가 정확하게 검출된 것입니다.

Recall (R): 0.98
→ 실제 객체 중 약 **98%**를 검출했음을 의미합니다.

mAP50: 0.902
→ IoU=0.5에서의 평균 정밀도가 **90.2%**임을 나타냅니다. 일반적으로 YOLOv5의 핵심 성능 지표입니다.

mAP50-95: 0.745
→ **IoU 0.5부터 0.95까지의 평균 mAP가 74.5%**입니다. 이 값은 더 엄격한 조건에서 모델의 성능을 평가하는 기준입니다.

3. 세부 결과
클래스별로 P, R, mAP50, mAP50-95 값이 표시되어 있습니다.
P, R: 정밀도와 재현율. 클래스별 성능을 평가합니다.
mAP50: 클래스별 IoU=0.5 조건에서의 성능.
mAP50-95: IoU를 0.5부터 0.95까지의 평균 성능입니다.
최종 결과는 모든 클래스를 종합한 수치로 나타나며, 이를 기준으로 모델의 성능을 평가합니다.

4. 최종 결과
mAP50이 90.2%로 높은 정확도를 보이고 있습니다.
mAP50-95는 74.5%로, 조금 더 까다로운 조건에서도 모델이 좋은 성능을 보여줍니다.
Precision(정밀도)은 61.5%로 검출된 객체 중 일부는 잘못된 검출이 포함될 수 있음을 의미합니다.
Recall(재현율)은 98%로, 대부분의 실제 객체를 검출한 것으로 평가됩니다.
요약
학습된 모델은 **정확한 객체 검출 성능 (mAP50 90.2%)**을 보여주고 있으며, 엄격한 기준에서도 (mAP50-95) 약 74.5%로 평가됩니다.
Precision은 상대적으로 낮을 수 있으나 Recall이 높기 때문에 실제 객체를 많이 찾아내는 모델로 해석됩니다.
이 결과를 바탕으로 객체 검출 결과를 더 세밀히 개선하거나 활용할 수 있습니다.
